raster commits
====

- [Rasterize](http://en.wikipedia.org/wiki/Raster_graphics) an image in the contribution timeline ( on github ) generating fake commits.
- `bash` + `git` + `date +%Y%m%d -s "20081128"`

image : [lenna](http://www.cs.cmu.edu/~chuck/lennapg/)
